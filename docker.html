Docker

Docker

1、常用命令

$ docker info 
$ docker pull [images]      //下载镜像
$ docker push [images]      //提交镜像
//查看
$ docker images     //查看镜像
$ docker ps -a      //查看容器

//删除
$ docker rmi [images]        //删除镜像
$ docker rm [container]     //删除容器

//启动、暂停、重启
$ docker start [container]
$ docker stop [container]
$ docker restart [container]

$ docker run 
$ docker run
    --name [NAME]
    -v /home/Docker:/hotyan     //映射到宿主机目录/home/Docker
    -p 8888:80      //端口映射
    -d      //允许直接返回
    [images]    //镜像名
