PHP语法

  • 所有用户定义的函数，类和关键字大小写不敏感

    • if
    • else
    • echo
  • 变量，大小写敏感


PHP变量规则

  1. 只能包含：数字、字母、下划线
  2. $ 开头，接变量名：第一个字字符必须是字母或下划线，不能是数字
  3. 大小写敏感

JS变量规则

  1. 只能包含：数字、字母、下划线、$
  2. 第一个字符必须是：字母或下划线或 $
  3. 大小写敏感


PHP 变量作用域

  1. local

  2. global 用于函数内访问全局变量

  3. static 保留局部变量不被删除

  4. parameter


PHP echo 和 print

  • echo： 可以输出一个或多个字符串
  • print：只能输出一个字符串，返回值总为1


PHP EOF

以<<<EOF 开始标记开始，以EOF结束标记结束(顶头写，不缩进和空格)，末尾必须有分号;

PHP 字符串变量

  1. 并置运算符(./,)

     <?php 
 $txt1="Hello world! "; 
 $txt2=" What a nice day!"; 
 echo $txt1 .  $txt2;    //Hello world!  What a nice day!
 ?>

  2. strlen() 返回字符串的长度:

     strlen('hello world!');  //12

  1. strpos() 匹配指定字符或文本，返回字符位置，否则false:

     strpos('hello world!','world');    //6


PHP魔术常量

  • _LINE_:依赖于在脚本中所处的行来决定
  • _FILE_:文件的完整路径和文件名
  • _DIR_:文件目录
  • _FUNCTION_:函数名称，4.3新增，5之后区分大小写
  • _CLASS_:类名称，4.3新增，5之后区分大小写
  • _TRAIT_:trait的名称，代码复用的一个方法，5.4.0新加
  • _METHOD_:类的方法名，5.0.0新加，区分大小写
  • _NAMESPACE_:当前命名空间的名称，5.3.0新增，区分大小写

PHP文件处理

  • 打开指定文件fopen(file_name,打开模式r|w|x);

     <?php
  $file=fopen("hotyan.txt","r") or exit("Unable to open file!");//以只读打开hotyan.txt文件
  ?>

  • 关闭文件fclose($file);

      <?php
  $file = fopen("hotyan.txt","r");
  //do something
  fclose($file);//关闭前面打开的文件
  ?>

  • 逐行读取文件fgets($file);

      <?php
  $file = fopen("hotyan.txt","r") or exit("无法打开文件!");
  while(!feof($file)){//读取文件每一行，直到文件结尾
          echo fgets($file)."<br/>";
  }
  fclose($file);
  ?>

  • 逐字符读取文件fgetc($file)

      <?php
  $file = fopen("hotyan.txt","r") or exit("无法打开文件!");
  while(!feof($file)){
          echo fgetc($file);
  }
  fclose($file);
  ?>
