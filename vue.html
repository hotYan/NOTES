<script>
export default {
 import PhoneSet from "./phone";
 data(){
     let validateName = (rule,value,callback)=>{
         let reg = /[^a-zA-Z0-9_.\-\u4e00-\u9fa5]/;
         if(value){
             if(value.length > 50){
                     callback(new Error('最大长度为50个字符'));
                 }else if (reg.exec(value)) {
                     callback(new Error('包含非法字符(应由数字、字母、中文、_.-组成)'));
                 } else {
                     callback();
                 }
             }
         }
     return{
         ruleForm:{
             email: '',
             name: '',
             phone: [{tell:''}]
         }
         formRules:{
             email:[{ 
                     required: true, 
                     message: '请输入邮箱地址', 
                     trigger: 'blur' 
                 },{ 
                     type: 'email', 
                     message: '请输入正确的邮箱地址', 
                     trigger: ['blur', 'change']
                 }
             ],
             name:[{
                     require: true,
                     message: '请输入昵称', 
                     trigger: 'blur' 
                 },{
                     pattern:/[^a-zA-Z0-9_.\-\u4e00-\u9fa5]/, 
                     message: '包含非法字符(应由数字、字母、中文、_.-组成)',
                     trigger: ['blur', 'change']
                 },{
                     max:20, 
                     message: '最大长度为20个字符',
                     trigger: 'change'
                 },{ 
                     validator:validateName, 
                     trigger: ['blur', 'change']
                 }
             ]
         }
     }
 },
 methods:{
     //删除
     deletePhone(index) {
         this.ruleForm.phone.splice(index, 1)
     },
     //添加
     addPhone() {
         this.ruleForm.phone.push( { tell: "" } )
     },
 }
}
</script>

Vue 双向绑定原理[2019.12.04]

通过数据劫持结合发布者-订阅者模式来实现的。

  1. 数据劫持

    通过Object.defineProperty()来实现数据劫持：

     var Hotyan = {}
 var age = 18
 Object.defineProperty(Hotyan,'age',{
     set: function(value){
         age = value
         console.log('你修改后的年龄是：' + value)
     },
     get: function(){
         return '你的年龄是：' + age
     }
 })
 Hotyan.age = 20     //你修改后的年龄是：20
 console.log(Hotyan.age)     //你的年龄是：20

    通过Object.defineProperty( )设置了对象Hotyan的age属性，对其get和set进行重写操作，顾名思义，get就是在读取age属性这个值触发的函数，set就是在设置age属性这个值触发的函数。

  2. 实现思路

    实现MVVM主要包含两个方面，data变化更新view，view变化更新data。

    view更新data,通过事件监听即可，关键点在于data改变，如何更新view：首先重点是如何知道data变了，通过Object.defineProperty()对属性设置一个set函数。当data被重新赋值,就会触发set函数。因此将一些需要更新view的方法写在set函数里面，data改变时就触发set函数中的更新操作，从而改变view。

  3. 实现过程

    实现数据的双向绑定，首先需要对数据进行劫持监听，设置一个监听器Observer监听所有属性。如果属性发生了变化，就告诉订阅者Watcher是否更新view。需要一个消息订阅器Dep来专门收集这些订阅者，然后统一管理监听器Observer和订阅者Watcher。接着需要一个指令解析器Compile对每个节点元素进行扫描和解析，将相关指令对应初始化为一个订阅者Watcher，并替换模板数据或者绑定相应的函数。此时，当订阅者Watcher接收到相应的属性变化，就会执行相应的更新函数，从而更新view。3个步骤：

    • 实现一个监听器Observer,劫持并监听所有属性，如果有变动，通知订阅者Watcher;
    • 实现一个订阅者Watcher，可以收到属性的变化通知并执行相应的函数，从而更新视图;
    • 实现一个解析器Compile，可以扫描和解析每个节点的相关指令，并根据初始化模板数据以及初始化相应的订阅器;
